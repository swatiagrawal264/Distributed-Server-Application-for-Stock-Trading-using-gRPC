1) How to run Part 1 and PArt 2 codes on edlab

log into edlab

2) Important points to keep in mind while running 

* Replace the Host with IP address of the current machine on both client and server.

* The IP address should be identical on both the client and server machines.

* Incase the Error message is shown, change the port number and run again.
